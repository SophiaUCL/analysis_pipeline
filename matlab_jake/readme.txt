Folder: matlab_Jake

MATLAB files that Jake used for population sink analysis.


Files:
calculateMeanDirections_combinedVectorsV4 - Main file to find the Population sinks

relativeDirectionFunction - called by the main function (see above) to calculate the MRL

getRelDirDist - called by file above to get the relative hd distribution
