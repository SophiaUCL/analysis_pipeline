Folder: HCT_analysis/plotting

Folder containing files that make plots for the HCT task

Specific files: 

make_maze_plots: called by spatial_processing_pipelins. Has functions that save plots into {derivatives_base}/analysis/maze_behaviour. Plots:
1. All start platforms (start_platforms.png)
2. The proportion of correct choices for each platform (proportion_correct.png)
3. Number of time each platform was visited (occupancy.png)
